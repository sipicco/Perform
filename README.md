> # It may not *Excel*, but it still performs!

Perform is a personal learning project aimed at recreating the core functionality of Microsoft Excel on a smaller scale.

The goal of this project is to explore the design and implementation of spreadsheet software, focusing on spreadsheet architecture, formula evaluation, data structures, and efficient rendering.

The application is built in C# and uses Raylib for high-performance rendering of the spreadsheet grid and cell interactions.

## Features

- [x] Spreadsheet-style grid interface
- [x] Cell rendering powered by Raylib
- [ ] Cell selection and navigation
- [ ] Text and numeric cell editing
- [ ] Basic formula parsing and evaluation


## Technologies

- C#
- .NET
- Raylib-cs

## Purpose

This project was developed as a personal study exercise to deepen my understanding of:

- Desktop application development
- Rendering pipelines
- Data-oriented design
- Expression parsing and evaluation
- Software architecture
- Performance optimization
